Example to run local version of the SimpleDreamer Backend Shower API server and frontend in a docker swarm environment

1. This example assumes you have docker and docker-compose installed

2. This example assumes you have a local registry running in docker swarm.
If you don't then use this project:
https://github.com/inventivejon/local_docker_swarm_registry

3. This example assumes you have a docker swarm running.
